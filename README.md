# Dashboard-project-mobile-sales-data
# Project Overview

This project is a Mobile Sales Dashboard created using Microsoft Power BI. The dashboard transforms raw mobile sales data into interactive visual insights that help understand sales performance, customer behavior, and business trends.

The project includes:

Raw sales dataset in Excel format
Interactive Power BI dashboard file

# Tools & Technologies Used
Microsoft Power BI – Dashboard creation & visualization
Microsoft Excel – Raw data storage and cleaning

# Dashboard Features

The dashboard provides insights into:

✅ Total Sales Analysis
✅ Monthly Sales & Quantity Tracking
✅ Customer Ratings Breakdown
✅ Transactions by Payment Method
✅ City-wise Sales Performance
✅ Mobile Model-wise Sales Comparison
✅ Day-wise Sales Trends
✅ Interactive Filters & Visualizations
